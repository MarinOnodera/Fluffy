# もこもこフレンド — キャラクター画像

`/chat` 画面などで使う 5 体の画像を、以下のファイル名でこのフォルダに置いてください。
1024x1024 程度の正方形 PNG/JPG を想定しています。

| ファイル名      | 表示名 | 雰囲気                    |
| --------------- | ------ | ------------------------- |
| `buddy-1.png`   | モモ   | ピンクの小さなまんまる    |
| `buddy-2.png`   | ソラ   | 水色×紫のふわふわ         |
| `buddy-3.png`   | ペシュ | 薄ピンクのまるい子        |
| `buddy-4.png`   | ルミ   | ラベンダーのおおきな目    |
| `buddy-5.png`   | ユキ   | まっしろもこもこ          |

画像が未配置の場合は `components/FluffyAvatar.tsx` の CSS フォールバック
(やわらかいグラデの丸 + 目 + 口) が描画されます。アプリは画像なしでも動作します。
